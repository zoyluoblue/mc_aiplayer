# AIBot — TikTok Promo (Remotion)

A 30-second, 9:16 vertical promo for **AIBot**, an autonomous Minecraft AI player.
All on-screen text is English. Source attribution: **zuuzii.com**.

- Format: 1080 × 1920, 30 fps, 900 frames (30s)
- Engine: [Remotion](https://remotion.dev) (programmatic React video)
- Visuals: motion graphics in a clean voxel aesthetic — **no external footage required**.
  (If you have real gameplay clips, drop them into a `<Sequence>` slot per scene.)

## Run

```bash
cd promo/tiktok
npm install
npm start          # opens Remotion Studio (live preview)
npm run render     # -> out/aibot-promo.mp4 (h264)
npm run still      # -> out/poster.png (a single frame for a thumbnail)
```

## Voiceover (natural male, open-source TTS)
The promo ships with a synced **male English voiceover** generated by
[Kokoro-82M](https://huggingface.co/hexgrad/Kokoro-82M) (open-source, runs locally),
voice `am_michael`. One clip per narration line lives in `public/vo/lineN.wav` and is
played at each scene's start (`VO` array in `src/Video.tsx`) so speech stays in sync
with the on-screen captions.

Regenerate / change the voice:
```bash
# one-time env (open-source, ~CPU):
uv venv --python 3.12 /tmp/ttsenv
uv pip install --python /tmp/ttsenv/bin/python kokoro "misaki[en]" soundfile numpy
# generate (edit VOICE / lines in tts/gen_voiceover.py first if you like):
VIRTUAL_ENV=/tmp/ttsenv /tmp/ttsenv/bin/python tts/gen_voiceover.py
```
Male voice options: `am_michael`, `am_adam`, `am_onyx` (US) · `bm_george`, `bm_lewis` (UK).

## Optional music bed
Drop a royalty-free track at `public/music.mp3`, then uncomment the music `<Audio>`
line in `src/Video.tsx` (keep volume ~0.18 so it sits under the voice).
